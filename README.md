# Dockerfile for bluemixcli on ubuntu

Dockerfile installs cloudfoundry cli, bluemix cli, docker plugin(ci), and docker cluster(cs) plugins. It also install kubernetes cli.

# references
- [cloudfoundry cli install](https://docs.cloudfoundry.org/cf-cli/install-go-cli.html) - Install cf command.
- [bluemix command install](http://clis.ng.bluemix.net/ui/home.html) - Install bluemix(bx) command.
- [docker install](https://docs.docker.com/engine/installation/linux/ubuntu/#install-using-the-repository) - Install docker client.
- [container plugin install](https://console.ng.bluemix.net/docs/containers/container_cli_cfic_install.html#container_cli_cfic_install) - Install docker plugin.
- [container-cluster plugin install](https://console.ng.bluemix.net/docs/containers/cs_cli_devtools.html) - Install kubenetes plugin. 
- [kubenetes cli install](https://kubernetes.io/docs/tasks/tools/install-kubectl/) - Install kubctl command.
